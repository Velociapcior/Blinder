Blinder

A conversation-first blind dating app. This kit packages the components, tokens, and copy rules that keep the product calm, equal-weight, and anti-urgent. Photos are hidden until mutual intent — the UI should be too.

Colors

Warm Dusk palette

Warm neutrals anchor everything. One muted plum primary, one ceremonial amber --reveal reserved exclusively for the reveal moment. No high-saturation colour appears in trust-critical flows.

Typography

Lato · generous line-height

A single family, four weights. Line-height 1.6–1.65 on body keeps conversation breathable. Sentence case in every heading; letter spacing stays at 0 except buttons and captions.

Spacing

8-pt grid, minimal elevation

Space tokens from 4 to 48. Shadows reserved for the single primary CTA, elevated modals, and the reveal portrait's amber glow. Depth isn't used to imply hierarchy — that work is done with whitespace.

Components

Components

Every custom component used in the product. The visual vocabulary is narrow on purpose — recurring use of the same few primitives builds the calm signature.

Screens

Screens in context

The same components, assembled into the five core screens. Use these as reference compositions — single column, generous padding, quiet navigation.

Motion

Two speeds

Utility transitions are near-instant (120–180ms). Emotional moments are slow, emphasized, and unhurried. The gap between the two is part of the design.
